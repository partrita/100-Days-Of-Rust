## 단어 찾기

2D 보드와 사전의 단어 목록이 주어지면 보드에서 모든 단어를 찾으십시오.

각 단어는 순차적으로 인접한 셀의 문자로 구성되어야 하며, "인접한" 셀은 가로 또는 세로로 이웃한 셀입니다. 동일한 문자 셀은 한 단어에서 두 번 이상 사용할 수 없습니다.

### 예시

```text
입력:
board = [
  ['o','a','a','n'],
  ['e','t','a','e'],
  ['i','h','k','r'],
  ['i','f','l','v']
]
words = ["oath","pea","eat","rain"]

출력: ["eat","oath"]
```

---

### 참고:

- 모든 입력은 소문자 `a-z`로 구성됩니다.
- `words`의 값은 고유합니다.

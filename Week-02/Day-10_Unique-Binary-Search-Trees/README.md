## 고유한 이진 검색 트리

`n`이 주어졌을 때, 1부터 `n`까지의 값을 저장하는 구조적으로 고유한 **BST**(이진 검색 트리)는 몇 개일까요?

### 예시

```text
입력: 3
출력: 5
설명:
n = 3이 주어지면 총 5개의 고유한 BST가 있습니다:

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
```

### 제약 조건

- `1 <= n <= 19`

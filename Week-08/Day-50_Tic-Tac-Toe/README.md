# 틱택토

_틱택토_ 게임에서 문자 입력 배열을 받는 함수를 만드십시오. 입력은 플레이어 1로부터 **"X"**로, 플레이어 2로부터 **"O"**로, 빈칸은 **"#"**으로 받습니다. 프로그램은 승자 또는 무승부 결과를 반환합니다.

## 예시
```text
ticTacToe([
  ["X", "O", "O"],
  ["O", "X", "O"],
  ["O", "#", "X"]
]) ➞ "Player 1 wins"

ticTacToe([
  ["X", "O", "O"],
  ["O", "X", "O"],
  ["X", "#", "O"]
]) ➞ "Player 2 wins"

ticTacToe([
  ["X", "X", "O"],
  ["O", "X", "O"],
  ["X", "O", "#"]
]) ➞ "It's a Tie"
```
